# ZiiOZ 🌐

ZiiOZ is a next-generation social platform blending AI-powered content creation, real-time engagement, and business-friendly features — all built for creators, businesses, and communities.

## 🚀 Features

- 🎨 AI-powered content creation
- 💬 Real-time reactions and messaging
- 🛍️ ZiiOZ Shop integration
- 🔐 Privacy-first with GDPR readiness
- 🌍 Multi-language support
- ⚡ Fast, modern Vite + React + TypeScript stack

## 🧪 Tech Stack

- [React 18](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Wouter](https://github.com/molefrog/wouter) (lightweight routing)
